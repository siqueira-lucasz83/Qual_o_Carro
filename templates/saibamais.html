Informações do Projeto

Conheça em profundidade o funcionamento, estrutura e propósito do Qual é o Carro?

Visão Geral

Qual é o Carro? é um jogo web interativo onde o jogador tenta adivinhar o modelo de um carro com base em dicas progressivas. A cada tentativa errada, uma nova dica é revelada — como marca, ano, potência e carroceria — até que o jogador acerte ou esgote suas chances. O sistema pontua o jogador e registra seu desempenho em um ranking global.

Funcionalidades

  • Jogo de adivinhação: com dicas automáticas e sistema de pontuação baseado em acerto e número de tentativas.
  • Cadastro e login de jogadores: com validação de campos e autenticação segura.
  • Ranking: exibe os melhores jogadores com rolagem arcade e destaque para os top 5.
  • Validação de carros: IA que analisa dados técnicos e sugere correções para inconsistências.
  • Interface retrô pixelada: inspirada em fliperamas dos anos 80, com fonte arcade e botões estilizados.
  • Modo administrador: permite cadastrar, editar e validar carros diretamente no sistema.
  • Feedback visual: mensagens de acerto e erro com cores vibrantes e animações leves.
  • Sistema de dicas dinâmicas: cada dica é revelada progressivamente com base nas tentativas restantes.
  • Controle de sessão: o sistema mantém o jogador logado e rastreia sua pontuação atual.

Tecnologias Utilizadas

  • Python + Flask: estrutura principal do backend, gerenciamento de rotas, sessões e lógica do jogo.
  • HTML + CSS (retrô pixelado): estrutura visual com estilo inspirado em jogos arcade.
  • JavaScript: controle de tentativas, exibição de dicas, manipulação de DOM e requisições assíncronas.
  • MySQL: banco de dados relacional para armazenar jogadores, pontuações e informações dos carros.
  • Jinja2: motor de templates para renderizar páginas dinâmicas com dados do backend.
  • Validador de dados com IA: sistema inteligente que verifica se os dados dos carros cadastrados são válidos e sugere correções.
  • Fontes e ícones: uso da fonte “Press Start 2P” e emojis para reforçar o estilo retrô.

Arquitetura do Sistema

O projeto segue uma arquitetura MVC simplificada:

  • Model: Representa os dados dos jogadores e carros, conectados ao MySQL.
  • View: Templates HTML com Jinja2 que exibem as páginas do jogo, ranking e administração.
  • Controller: Rotas Flask que recebem requisições, processam lógica e retornam respostas.

Além disso, o sistema usa JavaScript para interações em tempo real e AJAX para comunicação assíncrona com o backend.

Segurança

  • Validação de campos: todos os inputs são verificados no frontend e backend.
  • Controle de sessão: o Flask gerencia sessões para manter o jogador logado com segurança.
  • Proteção contra manipulação: pontuações e dados são validados no backend antes de serem salvos.

Objetivo do Projeto

O projeto tem como objetivo unir entretenimento com aprendizado técnico. Ele explora conceitos de:

  • Desenvolvimento web com Flask
  • Integração com banco de dados relacional
  • Design de interfaces retrô e responsivas
  • Validação inteligente com IA
  • Gamificação e ranking competitivo

É uma vitrine para quem curte carros, tecnologia e nostalgia digital.
